# A. Nezzar and Colorful Balls

### 题意

给定不下降的$n$个数，现要给这$n$个数染色，规则是相同颜色的数必须要严格递增，求最少需要的颜色数。

### 思路

对于一个严格递增的数列，很明显只需要将它们都染成一种颜色就可以满足题意，而题目保证数列不下降，说明数列除了严格递增的情况还有可能在一段上相等，而对于这相等的若干个数我们只能将其染上不同的颜色，由此问题就变成了求一段不下降序列的最长相等字串，而这个是很容易实现的。

# B. Nezzar and Lucky Number

### 题意

给定$n$个小于$10^9$的数，请分别判断这$n$个数能否表示为若干个十进制下包含数字$q(1 \leq q \leq 9)$的正整数的和。

### 思路

以$q=7$为例，我们可以发现包含数字$7$的数有$7,17,27\cdots67,70,71\cdots79,87,97$，注意从$70$到$79$我们有连续的$10$个数都包含数字$7$，那么假设要判断一个数字$x$是否能表示为若干个包含数字$7$的正整数的和，当$x>79$时，我们发现$(x-70)-(x-79)=9>7$，这就说明了在$(x-79)$到$(x-70)$里必定有一个数会被$7$整除，那么这个数就可以表示为若干个$7$相加，从而满足题意，换言之任何一个大于$69$的数都可以满足条件。接下来只需要判断小于$69$的数有哪些满足条件，至于这一点就不是很难实现。下一步我们将其推广到一般，由上可证任意大于$(q \times 10-1)$的数都可以满足，而小于等于$(q \times 10-1)$的数我们可以用完全背包和埃氏筛的思想（虽然我更喜欢称其贴邮票），如果数$x$可以表示为若干个十进制下包含数字$q(1 \leq q \leq 9)$的正整数的和，那么我们令$dp[x]=1$然后对任意$dp[y]=1,dp[x+y]=1$，最后只需要判断一个小于等于$(q \times 10-1)$的$dp$值就可以了。

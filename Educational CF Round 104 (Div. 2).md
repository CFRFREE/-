# B. Cat Cycle

### 题意

有两只猫和编号为$n$个点，一开始，A猫在$n$号点，B猫在1号点。每过一个小时A猫向编号小的一个点移动，B猫向编号大的一个点移动，即A猫沿着 $n,n−1,n−2,…,3,2,1,n,n−1$,B猫沿着$1,2,3,…,n−1,n,1,2,… $的顺序移动。如果在某一个时刻A猫和B猫都想移动到某个点那么B猫会选择将这个点让给A猫，自己移动到下一个点。求第$k$个小时B猫在哪个点$k (2≤n≤10^9; 1≤k≤10^9)$。

### 思路

首先讨论$n$，发现如果$n$是偶数那么不存在某个时刻两只猫都想移动到同一点，所以只要计算$k\bmod n $并稍作处理即可；当$n$是奇数时，通过手玩数据可以归纳出B猫所在的点的编号是$\dfrac{n-1}{2}$个公差为$\dfrac{n+1}{2}$的等差数列交替排列而成的，由此便可以$O(1)$得出每一次的结果

# C. Minimum Ties

### 题意

有$n (2≤n≤100) $只球队，每只球队都要与其他$n-1$只球队进行一场比赛，比赛的胜者得3分，负者不得分，若平局则两队各得一分，请给出一种比赛结果，使得在平局最少的情况下每支队伍的得分相同。

### 思路

讨论$n$，不难发现当$n$为奇数的时候，每只队伍都要进行$n-1$场比赛，又因为$n-1$为偶数，所以我们可以安排每只队伍胜负各$\dfrac{n-1}{2}$场，这样便可以保证在平局数最少（为0）的情况下各队的得分相同。当$n$为偶数时，我们可以安排每支队伍的$n-1$场比赛中有一场平局，其他的胜负各$\dfrac{n}{2}-1$场，这样易证每支队伍的得分都是相同的，至于如何证明此时平局的场数是最小的本人还没有严格地证明，不过大概脑补一下应该确实如此。至于如何安排每场比赛的胜负，本来想定义一个二维数组记录的，后来发现可以手玩数据找通解，反正题目也是要求给出一种合法的结果就行（这类构造题基本上都是这么处理的），至于通解的情况每个人的找法都不同，这里就不详细叙述了。

# D. Pythagorean Triples

### 题意

共$t (1≤t≤10^4)$组数据，每组数据给定一个$n,1≤n≤10^9$,求满足下列关系的三元组$(a,b,c)$的个数

$\begin{equation}
\left\{
             \begin{array}{lr}
             a ^ 2 + b ^ 2 = c ^ 2 \\
             c=a ^ 2 - b\\
             1 \leq a \leq b \leq c \leq n 
             \end{array}
\right.
\end{equation}$

### 思路

计算上述式子可得$c=\dfrac{a^2 +1}{2} \leq n$，所以$a \leq \sqrt{2n-1}$，再通过打表找规律我们发现$a$是以3为首项，2为公差的等差数列，又因为$a$固定时$b,c$也固定，所以三元组的个数即为$\dfrac{\sqrt{2n-1}-3}{2}+1$
